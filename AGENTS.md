# Strata — 意味グラフ文書フォーマット

## 概要
人志向ドキュメントと機械志向ドキュメントを1つの canonical source から派生させる文書表現。
Rust で実装。リポジトリは設計対話から起こした初期成果物。

## アーキテクチャ
- `crates/` : Rust クレート群
- `docs/` : 設計ドキュメント

## 環境
- Rust（Cargo）
- Nix flake で宣言的に管理

## ルール
- commit / push / merge はユーザーの指示がないと絶対に実施しない
- テスト時は実装を触らない、実装時はテストを触らない
- できるだけ nix で宣言的にする
- 設計書・プランは `docs/` ディレクトリに作成

## 作業ディレクトリ
このクローンは `~/dev/strata/` にあり、自由に編集可能。
